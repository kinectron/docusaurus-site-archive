Bring Your Body Online

Kinectron sends Kinect depth, color and skeletal data over a peer network. It can be used in a number of ways.

Kinect in Browser

Receive real-time Kinect streams in the browser, including in creative coding libraries like p5.js and three.js.

Broadcast Kinect

Broadcast Kinect data across local or global networks to one (or many) users.

Combine Kinect Streams

Combine Kinect streams from more than one locations in one single webpage.
